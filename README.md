# VUE SSR BASIC

### Descripción

Característica básica de Server Side Render en Vue | Js | Node

```sh
# 1) Clonar repositorio
$ git clone -
# 2) Ubicarse en el directorio
$ cd vue-ssr
# 3) Instalar dependencias del proyecto
$ npm install
# 4) Iniciar proyecto
$ npm start
```
